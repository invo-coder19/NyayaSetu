# NyayaSetu Frontend

React + Vite application for the NyayaSetu portal.

## Features

- Victim Portal - Apply for Relief
- Officer Portal - Case Review & Grievances Management  
- Financial Portal - Fund Tracking
- Ministry Dashboard - Analytics
- Transparency Hub - Public Data

## Tech Stack

- React 18
- Vite
- TailwindCSS
- Shadcn UI
- Lucide React Icons

## Development

```bash
npm install
npm run dev
```

Access the application at `http://localhost:5173`

## Build

```bash
npm run build
npm run preview
```
